# Pepagora Email Templates

Collection of responsive HTML email templates for the Pepagora B2B marketplace platform.

## Templates

### Welcome Emails
- `welcomeMail.html` - Supplier welcome email
- `buyerWelcome.html` - Buyer welcome email
- `bothWelcome.html` - Welcome for both buyer and seller

### Educational Emails
- `educationalOne.html` - Dashboard features guide
- `educationalTwo.html` - How Pepagora works
- `index.html` - Product listing guide
- `sellOffer.html` - Sell offers setup guide

### Engagement & Reactivation
- `engagementOne.html` - Support ticket confirmation
- `engagementTwo.html` - Support ticket notification (admin)
- `reactivationOne.html` - Reactivation with tips
- `reactivationTwo.html` - Urgent missed leads alert

### Product Management
- `firstProduct.html` - Add first product
- `firstProductTwo.html` - Publish catalog
- `sourcingOne.html` - Sourcing encouragement (both)
- `sourcingTwo.html` - Sourcing encouragement (buyer)
- `incompleteBS.html` - Incomplete profile alert

### Transactional
- `otpmail.html` - OTP verification email

## Usage

1. Replace variables like `{Supplier Name}` with actual user data
2. Update dynamic content (ticket IDs, OTP codes, dates)
3. Ensure all links point to correct Pepagora URLs
4. Test in multiple email clients before sending

## Technical Details

- **Layout:** Table-based HTML for email compatibility
- **Width:** 440px max content width
- **Responsive:** Mobile-optimized with media queries
- **Brand Color:** #d92c27 (Pepagora Red)
- **Images:** Hosted on Pepagora CDN

- Images: `assets/img/`
